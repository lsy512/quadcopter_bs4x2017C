Remoter_bs2017hh_fwhv30_170301.hex is a remoter firmware, which support to update the remoter firmware via USB port.
1, the hex file MD5 code is: AA1F1F76924CFA2B21F67516C096FA78
2, when working at USB frmware update mode, the right-down corner switch in the remoter must be setted to BL
3, when working at control mode, the right-down corner switch in the remoter must be setted to the non-BL
